## Setup
- Add `quasar-nuxt` dependency using yarn or npm to your project
- Add `quasar-nuxt` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'quasar-nuxt',

    // With options
    ['quasar-nuxt', { /* module options */ }],
 ]
}
```
